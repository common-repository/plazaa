Das plazaa Plugin für WordPress bietet dir eine Anzahl an tollen Widgets mit deinen Bewertungen zu den besten Locations der Stadt von plazaa.de

== Description ==

Mit dem plazaa Plugin für WordPress kannst du dein plazaa-Profil, oder deine letzten Bewertungen von plazaa.de auf deinem Blog anzeigen lassen. 
Weitere Infos auf http://plazaa.de/

== Installation ==

1. Plugin herunterladen und in das Plugins-Verzeichnis auf Deinem Server kopieren.
2. Das Plugin aktivieren.
3. In den Einstellungen zum Plugin Deinen Plazaa-Benutzernamen eintragen.
4. Beliebig viele der neuen plazaa-Widgets zu Deinem Blog hinzufügen - Fertig!

== Frequently Asked Questions ==

= Es funktioniert nicht. Was kann ich tun? =

Das ist die erste Version unseres Plugins und auch wenn wir es auf einigen Blogs mit unterschiedlichen Versionen und Themes ausprobiert haben, kann es natürlich trotzdem sein, dass es nicht überall so reibungslos funktioniert. Schreibe uns dann doch einfach eine Mail an wordpress@plazaa.de und beschreibe, was nicht klappt. Am Besten natürlich mit der Angabe Deiner Wordpress-Version und dem verwendeten Theme.

== Screenshots ==

1. Zeige Deine Bewertungen von [Plazaa.de](http://plazaa.de "Die besten Locations der Stadt, von Freunden empfohlen")
2. Einfache Einbindung in Dein Blog über 3 verschiedene Widgets
3. Caching ist direkt mit eingebaut, ansonsten sind die notwendigen Einstellungen extrem simpel.
